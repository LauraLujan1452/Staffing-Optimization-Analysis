# Data Sources

Raw CSVs are not committed to this repo. To reproduce the analysis, download:

* **hotel_bookings.csv** — [Kaggle: Hotel Booking Demand](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand)
* **dly1823_subset_2015/2016/2017.csv** — [Met Éireann historical data](https://www.met.ie/climate/available-data/historical-data), station 1823
* **dublin_events_2015_2017.csv** — (AI Generated)

Place downloaded files in this folder before running `Staffing.ipynb`.
